# Data Field Notes

## `cards.yaml`

- `id`: card id
- `name`: card name
- `type`: machine / pilot / command / base
- `lv`: level
- `cost`: cost
- `ap` / `hp`: body stats
- `resonance.kind`: resonance type
- `resonance.values`: resonance parameters
- `text`: raw card text

## `value_table.yaml`

- `name`: effect name
- `param1_name`: parameter 1 name
- `param2_name`: parameter 2 name
- `outputs`: value-model outputs
- `notes`: interpretation notes

## `decks/*.txt`

Readable player-facing deck format:

```text
// Main Deck
4x ST01-005
4x ST05-004
```

## `output/*.xlsx`

Generated review files only, not source of truth.
